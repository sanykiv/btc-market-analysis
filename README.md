# BTC Market Analysis

Анализ рыночных данных BTC/USDT и ETH/USDT через биржевой API.  
Цель: выявить рыночные неэффективности и проверить торговые гипотезы на исторических данных.

**Стек:** Python, pandas, NumPy, matplotlib, ccxt, Jupyter

---

## Данные
- Источник: Binance REST API через библиотеку ccxt
- Инструменты: BTC/USDT, ETH/USDT
- Таймфрейм: дневные и часовые свечи (OHLCV)
- Период: 2022–2024

---

## Блок 1 — Сбор и подготовка данных
- Загрузка исторических OHLCV-данных через ccxt
- Очистка тайм-серии: пропуски, выбросы, корректность таймстемпов
- Агрегация и нормализация данных

## Блок 2 — Технические индикаторы
- RSI (Relative Strength Index)
- Скользящие средние (SMA, EMA)
- Уровни поддержки и сопротивления по локальным экстремумам

## Блок 3 — Бэктест гипотез
- Гипотеза 1: цена отбивается от уровня поддержки в статистически значимом числе случаев
- Гипотеза 2: ложные пробои уровней перед разворотом встречаются системно
- Оценка winrate и соотношения риск/доход
